# Reporte: Despliegue de sitio estático en AWS S3

## Objetivo
Demostrar un flujo DevOps básico con GitHub, EC2 y AWS S3.

## Metodología
1. Configuración de AWS CLI.
2. Creación del bucket y política pública.
3. Subida de archivos con `aws s3 sync`.
4. Automatización con `deploy.sh`.

## Resultados
Sitio desplegado en:
http://feysa-devops-site.s3-website-us-east-1.amazonaws.com

## Conclusión
Este ejercicio muestra cómo versionar código en GitHub y desplegarlo en la nube de forma rápida.  
Aprendí la importancia de las políticas de acceso y la automatización para evitar errores manuales.
